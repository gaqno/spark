## 📍 Overview

The project is a Vue.js application created with Nuxt.js framework. It provides a chat interface that utilizes the OpenAI API to generate helpful responses to customer questions. The application also includes various components such as a home page with hero, statistics, and grids, a chat component with user and assistant messages, a sidebar component with menu items and content area, and a reports component with graphs and charts. The project aims to provide an interactive and visually appealing user experience while helping users get quick and accurate answers to their questions.
